## 💁🏻 Rules

>1. 의미 없는 커밋은 하지 않는다.
>2. 커밋 로그를 명확히 작성한다.
>3. 누구나 이해할 수 있도록 작성한다.
>4. 거창하지 않아도, 내용이 많지 않아도 최대한 의미있는 내용만을 작성한다.
>5. 중요하다고 판단되는 내용은 블로그에도 정리해놓는다.
>6. 각 주마다 회고록을 작성한다.
